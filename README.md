# Steps — учёт тренировок

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-success?style=flat-square)](https://madmaxim22.github.io/steps)

Небольшое React-приложение для учёта тренировок: дата и пройденные километры. Записи по одной дате объединяются (километраж суммируется), список можно редактировать и удалять.

## Возможности

- **Добавление тренировок** — дата в формате ДД.ММ.ГГ и пройденные км
- **Объединение по дате** — при повторной записи на ту же дату километраж суммируется
- **Редактирование и удаление** записей
- **Сортировка** таблицы по дате

## Стек

- React 19
- Vite 7

## Запуск

```bash
# зависимости
npm install

# режим разработки
npm run dev

# сборка
npm run build

# превью собранного приложения
npm run preview
```
